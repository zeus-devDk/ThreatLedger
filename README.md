## What is ThreatLedger?
ThreatLedger is an open cybersecurity intelligence repository covering:
- Latest breaches
- New CVEs and exploit analysis
- Security tools & research
- Roadmaps for students and professionals
- Curated insights from 3000+ trusted sources

## Why This Exists
Cybersecurity knowledge should be:
- Free
- Open
- Practical
- Unbiased
